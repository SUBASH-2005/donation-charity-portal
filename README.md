# Donation and Charity Management Portal

A full-stack web application developed to manage charity campaigns and donations efficiently.

## Features
- Admin can create and manage charity campaigns
- Users can donate to active campaigns
- Real-time donation tracking
- Donation history with total amount collected
- Clean and responsive UI

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB
- Version Control: Git, GitHub

## Project Structure
models/ - MongoDB schemas
routes/ - API routes
public/ - CSS and frontend assets
views/ - HTML pages
server.js - Application entry point
